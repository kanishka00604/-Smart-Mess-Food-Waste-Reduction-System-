# Smart Mess & Food Waste Reduction System

## Quick Start

### 1. Backend
```bash
cd backend
npm install
# Create .env file (see backend/.env.example)
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm start
```

### 3. MongoDB
- Install MongoDB locally OR use MongoDB Atlas (free cloud)
- Set MONGO_URI in backend/.env

## Ports
- Frontend: http://localhost:3000
- Backend:  http://localhost:5000
