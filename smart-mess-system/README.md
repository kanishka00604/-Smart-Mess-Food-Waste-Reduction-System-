# 🍱 Smart Mess Food Waste Reduction System

A full-stack AI-powered mess management system designed to reduce food waste in college hostels and cafeterias by predicting meal demand, managing bookings, tracking waste, and improving food distribution efficiency.

---

# 📌 Project Overview

The **Smart Mess Food Waste Reduction System** is a web-based platform that helps colleges and institutions manage hostel mess operations efficiently.

The system allows:

* Students to book meals in advance
* Admins to monitor food consumption
* AI-based prediction of food demand
* Waste tracking and analytics
* Feedback collection from students
* QR-based meal verification

The main goal of this project is to:

✅ Reduce food wastage
✅ Improve mess management
✅ Increase operational efficiency
✅ Enhance student experience

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/kanishka00604/-Smart-Mess-Food-Waste-Reduction-System-.git
```

---

# 🔧 Backend Setup

## Navigate to Backend Folder

```bash
cd smart-mess-system/backend
```

## Install Dependencies

```bash
npm install
```

## Create Environment File

Create a `.env` file and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## Run Backend Server

```bash
npm start
```

---

# 🎨 Frontend Setup

## Navigate to Frontend Folder

```bash
cd smart-mess-system/frontend
```

## Install Dependencies

```bash
npm install
```

## Start Frontend

```bash
npm start
```

---

# 🤖 AI Service Setup

## Navigate to AI Service Folder

```bash
cd smart-mess-system/ai-service
```

## Install Python Dependencies

```bash
pip install -r requirements.txt
```

## Run AI Service

```bash
python app.py
```
